#  Текущий подход к разработке

Разработка начинается с проектирования архитектуры системы

Причина

Правильная архитектура — фундамент успешного проекта. На мобильной платформе это критически важно, так как ошибки на ранних этапах могут привести к:
-  Невозможности добавления новых экранов без поломки старых
-  Тормозам и лагам в UI из-за неправильной работы с потоками
-  Сложностям с поддержкой разных версий Android
-  Проблемам с сохранением состояния при повороте экрана
